✅ 2. HTML Usage Example

html
<h1>Get <span class="gradient-word">Support</span> Instantly</h1>

🧪 Troubleshooting Tips

  • Make sure the <span> is not nested inside a parent with overflow: hidden or backface-visibility applied.

  • Confirm the browser you're testing on supports -webkit-background-clip: text (most modern Chrome, Edge, Safari, and Android do).

  • If you're testing in Firefox: It requires -moz-background-clip: text which is not supported without a flag and is inconsistent. Consider using SVG fallback if Firefox support is critical.
